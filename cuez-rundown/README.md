# Cuez Rundown Agent
## Create & Activate Virtual Environment (Recommended):

```bash
python -m venv .venv
```
### macOS/Linux:

```bash
source .venv/bin/activate
```
### Windows CMD:
```bash
.venv\Scripts\activate.bat
```

### Windows PowerShell:
```bash
.venv\Scripts\Activate.ps1
```

## Install packages

```bash
pip install -r requirements.txt
```
